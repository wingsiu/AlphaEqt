# AlphaEqt Demo

## Run in Xcode (recommended)

```bash
cd /Users/alpha/Desktop/swift/AlphaEqt
open Demo/Package.swift
```

Then select the **DemoApp** scheme (not AlphaEqt) and press ▶ Run.

## Run from terminal

```bash
cd /Users/alpha/Desktop/swift/AlphaEqt/Demo
swift run DemoApp
```

The app window should appear in your Dock. If it doesn't come to the front automatically, click its icon in the Dock.

## What you'll see

- `E = mc²` — superscript rendering
- `a² + b² = c²` — multiple superscripts
- `x₁ + x₂ + xₙ` — subscripts
- `a_{ij}` — multi-character subscript with braces
- `xᵢ²` — combined superscript + subscript
- `x³ - 2x² + 5x - 7 = 0` — polynomial
